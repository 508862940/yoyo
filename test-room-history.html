🛠️ 技术实现详情

房间历史记录数据结构：
{
  roomId: string,
  nickname: string,
  timestamp: number,
  lastUsed: number
}
关键函数：
• loadRoomHistory() - 从localStorage加载历史记录
• saveRoomHistory() - 保存历史记录到localStorage
• addToRoomHistory() - 添加房间到历史记录
• updateRoomHistoryDisplay() - 更新UI显示
• formatMessageTime() - 格式化消息时间戳
存储机制：
• 使用localStorage持久化存储
• 最大保存10条历史记录
• 自动去重，相同房间只保留最新记录
• 按最后使用时间排序

🐛 已知限制和改进建议

当前限制：
  • 历史记录仅保存在当前浏览器中
  • 无法跨设备同步历史记录
  • 时间戳基于客户端时间，可能存在时区问题
未来优化方向：
  • 添加历史记录搜索功能
  • 支持历史记录分类（常用、最近等）
  • 添加房间描述或标签
  • 支持历史记录导出/导入
  • 云端同步历史记录
