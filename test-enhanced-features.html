🛠️ 技术实现详情

聊天记录数据结构：
{
  user: string,
  text: string,
  timestamp: number,
  isOwnMessage: boolean
}
房间历史记录增强结构：
{
  roomId: string,
  nickname: string,
  timestamp: number,
  lastUsed: number,
  category: string,
  tags: string[]
}
新增关键函数：
• loadChatHistory() - 加载聊天记录
• saveChatHistory() - 保存聊天记录
• addToChatHistory() - 添加聊天消息到历史
• loadChatHistoryToUI() - 加载历史消息到界面
• exportRoomHistory() - 导出历史记录
• importRoomHistory() - 导入历史记录
• updateRoomHistoryDisplay(searchTerm) - 支持搜索的显示更新
存储机制：
• 聊天记录：localStorage key: 'mqtt_chat_history'
• 房间历史：localStorage key: 'mqtt_room_history'
• 每个房间最多保存100条聊天记录
• 最多保存10条房间历史记录
• 导出格式：JSON with version control
