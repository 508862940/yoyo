🧪 MQTT聊天室在线人数功能测试

✨ 新功能介绍

在线人数显示：实时显示当前房间内的在线人数

在线用户列表：点击人数可展开查看所有在线用户

用户状态跟踪：自动跟踪用户的加入和离开

视觉区分：自己在用户列表中会特殊标注"(我)"

建议房间号：test-room-online

🛠️ 技术实现详情

数据结构：
let onlineUsers = new Set(); // 在线用户集合
let userJoinTimes = new Map(); // 用户加入时间记录
关键函数：
• addOnlineUser(username) - 添加在线用户
• removeOnlineUser(username) - 移除在线用户
• updateOnlineUsersDisplay() - 更新UI显示
事件处理：
• presence/join 消息 → 添加用户到在线列表
• presence/leave 消息 → 从在线列表移除用户
• 连接成功 → 添加自己，清空旧数据
• 断开连接 → 清空所有在线用户

🐛 已知限制和改进建议

当前限制：
  • 仅显示当前会话期间加入的用户
  • 无法显示连接前已在房间的用户
  • 依赖MQTT Last Will机制处理意外断线
未来优化方向：
  • 添加"谁在房间"查询机制
  • 显示用户在线时长
  • 用户头像或状态图标
  • 最大在线人数统计
